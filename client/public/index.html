How it works
Three steps. No private keys.
01

Connect & create token

Connect your TON wallet and generate a scoped token for each agent. Each token is an isolated session.

→
02

Agent requests transfer

Your AI agent calls the API or MCP tool with the destination and amount. The request is queued — nothing moves yet.

→
03

You approve & sign

Review the request in your dashboard. Approve it and sign with your wallet via TON Connect. Done.

Why Agent Gateway
Built for the AI-agent era
🔒

Zero key exposure

Agents never touch your private keys. They can only queue transfer requests that you sign.

⏱

Auto-expiring requests

Pending requests expire after 5 minutes. Tokens live until revoked. Clean control.

🛠

MCP + REST API

Native MCP server for Claude Code and Cursor. REST API with Swagger docs for everything else.

📋

Session isolation

Each token has a unique session. Multiple agents, each with their own scope — no cross-contamination.
